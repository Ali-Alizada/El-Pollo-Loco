Class: Chicken

Chicken(xopt)

new Chicken(xopt)

Creates a new Chicken instance. Loads walking and dead animations, sets random position, speed, and walk range. Animation starts after a random delay.
Parameters:
Name Type Attributes Default Description
x number | null <optional>
 null The horizontal start position. If null, a random position between 700 and 1200 is chosen.
