Class: Cloud

Cloud(xopt)

new Cloud(xopt)

Creates a new Cloud instance. Loads the cloud image and places the cloud at a given x position (or random if not provided). Automatically starts the cloud animation.
Parameters:
Name Type Attributes Default Description
x number | null <optional>
 null The horizontal position of the cloud. If null, a random x is chosen.
